2. 为什么这两个函数是等价的
```python
def subsets(nums):
    result = [[]]
    for num in nums:
        for element in result[:]:  # 每一轮循环都对result做了copy，如果不copy，会永远循环
            x = element[:]  # 切片器的拷贝是浅拷贝，如果不对里面的element做拷贝，result里原有的元素都会通过x的指向被改动
            x.append(num)
            result.append(x)

    return result
def subsets_2(nums):
    res = [[]]
    for num in nums:
        res += [ i + [num] for i in res] # 我也不知道什么原理，居然不会无限循环
    return res
nums = [1, 2, 3]
print(subsets_2(nums))
print(subsets(nums))
```
